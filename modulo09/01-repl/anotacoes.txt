REPL - READ EVAL PRINT LOOP
Ler executar e exibir valores de funções  
(Não recomendado para desenvolvimento mas para resolver problemas)

REPL é uma forma de debuggar o codigo 

para iniciar o processo basta fazer um node inspect index.mjs 

list(numero de linhas) - Mostra a quantidade de linhas do codigo desejada 
setBreakpoint(numero da linha) ou sb(numero da linha) - coloca o breakpoint na linha desajada 
cont - manda a execução continuar 
exec nomedavariavel - mostra o valor da variavel
clearBreakpoint(nomedoarquivo, linha) - limpa os breakpoints
next - continua para proxima linha 

repl - comando para iniciar
    nome da variavel - para mostrar o nome da variavel
